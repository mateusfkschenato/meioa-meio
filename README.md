# Meio a Meio 🥪

O **Meio a Meio** é uma plataforma criada para ajudar estudantes a encontrarem alguém para dividir um lanche de forma simples, rápida e econômica.

O projeto surgiu da observação de uma situação comum no ambiente universitário e escolar: muitas vezes uma pessoa gostaria de pedir um lanche grande, mas não tem com quem dividir. O Meio a Meio conecta estudantes que desejam compartilhar um pedido, reduzindo custos e evitando desperdícios.

## Funcionalidades

- 🔍 Encontrar outro estudante para dividir um lanche;
- 💬 Chat em tempo real após o pareamento;
- 👤 Identificação por nome e turma;
- 🔄 Cancelamento automático do pareamento quando um usuário sai do chat.

## Tecnologias

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Realtime Database
- Vercel
